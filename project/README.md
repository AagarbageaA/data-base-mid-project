# 課程管理系統 Course Management System

這個專案是我為「資料庫課程」設計的課程管理系統，目的是讓老師和學生能夠方便地管理、查詢大量課程資料。系統後端採用 MongoDB，前端則用 wxPython 製作桌面應用程式。所有資料都結構化儲存，方便查詢和維護。

## 專案動機與目標

在教育現場，課程資料常常需要彈性管理，例如依照年級、CEFR語言等級、課程系列等條件篩選。這個系統希望能解決：

- 如何快速新增、查詢、修改、刪除課程
- 如何讓使用者能夠用多種條件篩選課程
- 如何支援中文關鍵字的模糊搜尋

## 技術架構

- **Python 3.11**：主要程式語言
- **MongoDB**：NoSQL資料庫，適合儲存結構化課程資料
- **PyMongo**：Python 與 MongoDB 的橋樑
- **wxPython**：桌面GUI框架，讓操作更直覺

## 主要功能

1. **課程管理**：

- 新增課程（所有欄位皆可填）
- 編輯課程（直接在GUI修改）
- 刪除課程
- 查詢所有課程

2. **條件篩選**：

- 依照年級、CEFR等級、課程系列等條件篩選
- 支援多條件複合篩選

3. **模糊搜尋**：

- 可用中文或英文關鍵字搜尋課程名稱與摘要

4. **使用者介面**：

- 三大分頁：新增課程、管理課程、篩選/搜尋課程
- 操作簡單，結果即時顯示

## 資料結構設計

每筆課程資料都儲存在MongoDB的 `courses`集合，結構如下：

```json
{
  "_id": 課程ID (int),
  "name": {
   "full": "完整課程名稱",
   "short": "簡短名稱"
  },
  "summary": "課程摘要",
  "category": {
   "id": 類別ID,
   "name": "類別名稱",
   "path": "類別路徑",
   "path_names": "路徑名稱",
   "top_category": "頂級類別"
  },
  "level": {
   "id": "年級ID",
   "name": "年級名稱",
   "CEFR": "語言等級"
  },
  "series": "課程系列",
  "created_at": "建立時間",
  "updated_at": "更新時間"
}
```
